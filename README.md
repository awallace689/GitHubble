# RUN Express.js API
    > cd api
    > npm install
    > npm start

# RUN React.js Client
    > npm start
Navigate to https://localhost:3000

# About

Web app for comparing your GitHub profile to a sample of ~~300+~~ 800+ other GitHub users.

Made with the MERN (MongoDB, Express.js, React.js, Node.js) stack and GraphQL!
